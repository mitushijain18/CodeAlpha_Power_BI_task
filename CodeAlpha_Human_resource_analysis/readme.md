# Human Resource Analysis using Power BI

## Project Overview

This project is developed as part of the CodeAlpha Power BI Internship. The project focuses on analyzing Human Resource (HR) data to understand employee performance, workforce trends, and organizational insights using Microsoft Power BI.

The main objective of this project is to transform raw HR data into meaningful visual reports and interactive dashboards that help in making better HR decisions.

## About Human Resource Analysis

Human Resource Analysis involves collecting and analyzing employee-related data to understand workforce patterns, improve employee management, and support strategic decision-making.

Power BI helps HR teams analyze important factors such as employee demographics, department performance, salary distribution, employee satisfaction, and attrition trends.
